# Data Game Project

Analyse statistique et prévision de séries temporelles sur les émissions de gaz à effet de serre.

---

## Description

Ce projet analyse les données du Global Monitoring Laboratory sur les concentrations atmosphériques de différents gaz (CO2, CH4, N2O, SF6) pour étudier leurs tendances, leurs variations saisonnières et faire des prévisions.

---

## Jeux de données

- ch4_mm_gl.csv : Méthane (CH4)
- co2_mm_mlo.csv : Dioxyde de carbone (CO2)
- n2o_mm_gl.csv : Protoxyde d'azote (N2O)
- sf6_mm_gl.csv : Hexafluorure de soufre (SF6)

---

## Analyses réalisées

- Chargement et exploration des données
- Visualisation par nuages de points
- Décomposition additive : moyenne mobile, variations saisonnières, coefficients mensuels
- Droite de tendance par régression linéaire
- Prévisions sur les 24 prochains mois
- Décomposition multiplicative
